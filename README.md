# Drupal CMS AI SEO Recipe

This recipe sets up a Drupal site for AI-driven SEO optimization.

## Features
- **Metatag**: Comprehensive meta tag management.
- **Pathauto**: Automatic URL alias generation.
- **Simple Sitemap**: XML sitemap generation.
- **AI**: Integration with AI providers for content and metadata generation.

## Usage
Apply this recipe using the Drupal recipe runner or standard recipe commands.

```bash
php core/scripts/drupal recipe recipes/drupal-cms-ai-seo-recipe
```
